# Design System Theme

## Overview
This document outlines the core visual theme of our design system, reflecting a modern and user-friendly aesthetic. The theme is characterized by its use of a dark color scheme, balanced typography, and subtly rounded elements, aiming for clarity and approachability.

## Color Palette
The color palette is built around a harmonized set of colors, ensuring visual consistency and accessibility.

*   **Primary Color:** `#10a4ff` - A vibrant and trustworthy blue, used for key interactive elements, calls to action, and primary branding.
*   **Secondary Color:** `#2578d1` - A muted, supportive blue, ideal for secondary actions, borders, and less prominent UI elements.
*   **Tertiary Color:** `#8264ce` - A soft, elegant purple, serving as an accent for highlights, badges, or decorative purposes.
*   **Neutral Color:** `#61789d` - A medium gray providing a stable base for backgrounds, surfaces, and typography in the dark mode.

The overall color scheme is set to `dark` mode, providing a bright and airy interface.

## Typography
The typography strategy employs a consistent font family across various text roles for a cohesive look.

*   **Headlines:** `inter` - Modern and highly legible, perfect for conveying hierarchy and impact.
*   **Body Text:** `inter` - Ensures readability for long-form content and general interface text.
*   **Labels:** `inter` - Clear and concise for form fields, buttons, and other interactive labels.

## Shape and Form
The design embraces a subtle level of roundedness for UI elements.

*   **Roundedness:** `1` (Subtle roundedness) - Provides a touch of softness without compromising a clean, contemporary feel. Elements feature slightly softened corners.

## Spacing
Spacing is managed to provide a balanced and comfortable user experience.

*   **Spacing:** `2` (Normal) - Adheres to standard UI practices, offering sufficient whitespace to prevent clutter while maintaining information density.
